# Doss AI HR 开发任务清单（MVP）

文档日期：2026-02-26  
关联文档：`docs/PRD.md`、`docs/开发方案.md`

## 使用说明

1. 本清单按 M1~M4 分阶段执行。  
2. 每项任务默认要求：代码 + 测试 + 文档同步。  
3. 额外约束：**优先复用现有组件（`apps/desktop/src/components/Ui*.vue`）**。

1. M1 出口：四站抓取可跑通并入库。

## 组件复用约束（执行期必须遵守）

1. 新页面默认只允许使用现有 `Ui*` 组件拼装。  
2. 如必须新增组件，需先证明现有组件无法满足且不应通过小改达成。  
3. 禁止引入新的 UI 框架或组件库。
